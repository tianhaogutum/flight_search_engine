Welcome to the **Flight Information System** project! This project is a full-stack application that utilizes the Lufthansa API to provide flight information and booking capabilities.

## Tech Stack

- **Frontend**:
  - React
  - HTML
  - CSS
    
- **Backend**:
  - Java
  - Spring Boot
  - RESTful API
